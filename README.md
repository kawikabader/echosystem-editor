# Echosystem Editor

A browser-based parameter editor for the Empress Echosystem dual delay/reverb pedal, built with Web MIDI.

**[Open the Editor](https://kawikabader.github.io/echosystem-editor/)**

## Features

- Real-time MIDI parameter control for both engines
- Preset browser with save/load/export
- Tap tempo and MIDI clock sync
- Expression pedal assignment
- Routing configuration
- Undo/redo support
